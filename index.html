TopicPrep Mobile App

A gamified mobile learning app designed to boost student engagement and retention.

The Problem

Students found traditional test prep to be boring and unengaging. The challenge was to create a mobile-first experience that felt like a game but delivered real educational value.

My Role & Process

As the lead UI/UX designer, I was responsible for the entire design process, from user flows and wireframes in Figma to the final high-fidelity mockups and prototypes. We conducted user interviews and A/B tested two different quiz flows, which led to a 25% improvement in usability scores.

The Outcome

The final app design was praised for its intuitive interface and engaging features. This project helped demonstrate the power of a "gamified" approach to learning.

SkillMedha CMS Platform

A complex, multi-portal system for students, colleges, and corporate partners.

The Challenge

This was a large-scale project requiring distinct user experiences for three different user types (Students, College Admins, and Corporate Recruiters). The main challenge was to build a single, scalable design system that could serve all three portals while maintaining a consistent brand identity.

The Solution

I designed a component-based design system in Figma, which reduced design-to-development handoff time by 30%. The dashboard for admins was particularly successful, helping them manage student data more efficiently.
